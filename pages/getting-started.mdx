# Getting Started with Manim Web

## Installation

If you haven't already, install the Dart SDK
([tutorial](https://dart.dev/get-dart)).

You will also need to have <LaTeX/> installed.

Then, activate the `webdev` compilation tool by running:

```bash
dart pub global activate webdev
```
